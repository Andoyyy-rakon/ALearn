# ALearn - The Intellectual Sanctuary

ALearn is a sophisticated, AI-driven study ecosystem designed to transform passive reading into active mastery. It serves as a personal sanctuary for scholars who want to distill complex topics into structured, interactive learning materials.

![ALearn Logo](/frontend/public/logo.png)

## 🏛️ The Vision

In an age of information overload, ALearn provides the tools to simplify, categorize, and conquer any subject. By leveraging advanced Large Language Models, the system acts as a personal study assistant that understands context and generates pedagogical content tailored to the user.

## 🔄 How It Works

The ALearn workflow is designed to be seamless, intuitive, and mathematically precise:

### 1. The Entrance (Authentication)
Users enter their sanctuary via a secure **Google OAuth 2.0** gateway. This ensures that every scholar has a private, persistent deck that follows them across devices.

### 2. The Extraction (AI Topic Generation)
Scholars can input any subject—from "Quantum Mechanics" to "Philippine History." Our **Groq-powered AI engine** (Llama 3.1) processes the request to generate a structured set of flashcards. Each card is balanced for difficulty and pedagogical value.

### 3. The Mastery (Study Deck)
The **Study Dashboard** serves as the central hub. Users can interact with their cards using a 3D-perspective interface. This visual feedback loop is designed to improve retention and make study sessions feel "alive."

### 4. The Challenge (Quiz Master)
True mastery is tested in the **Quiz Master** module. The system generates dynamic, multiple-choice exams based on the scholar's chosen topic. 
- **Immediate Feedback**: Correct answers are celebrated.
- **AI Explanations**: For ogni incorrect answer, the AI provides a deep-dive explanation to ensure the scholar learns from their mistakes in real-time.

### 5. Manual Refinement
Scholars can also manually inject specific concepts into their sanctuary, allowing for a hybrid learning experience of AI-generated breadth and human-curated depth.

## 🎨 Design Philosophy

ALearn is built with a **Premium Minimalist** aesthetic:
- **Glassmorphism**: A sleek, modern UI that uses transparency and blur to create depth.
- **Micro-animations**: Powered by Framer Motion to provide tactile feedback on every click.
- **Adaptive Theme**: A professional "Midnight Sanctuary" dark mode for focused late-night sessions and a "Clean Scholar" light mode for daily productivity.

## 🛠️ Technological Foundations

- **Intelligence**: Groq Llama 3.1 8B Instant.
- **Frontend**: React 19, Tailwind CSS 4, Framer Motion.
- **Backend**: Node.js & Express Architecture.
- **Data Layers**: MongoDB & Mongoose.
